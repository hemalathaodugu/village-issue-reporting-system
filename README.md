# Village Issue Reporting System

## Project Description

Village Issue Reporting System is a web application designed to allow villagers to report local issues easily and help administrators manage and monitor those complaints efficiently.

Users can submit issues like road problems, water supply issues, street light failures, and garbage complaints. Admin can view and manage all reported issues through the dashboard.

## Features

### User Side

* Report Issue form
* Submit complaint details
* View confirmation after submission

### Admin Side

* Admin Dashboard
* View all reported issues
* Manage and track issue status

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS
* Vite
* Node.js

## Project Structure

* Frontend built using React
* Backend built using Node.js
* Data handling using Fetch API

## How to Run

1. Clone the repository
2. Install dependencies:

   ```
   npm install
   ```
3. Start frontend:

   ```
   npm run dev
   ```
4. Start backend:

   ```
   node server.js
   ```
